## 📱 About the Project

**Shalom-Salam** is a bidirectional language learning app that bridges **Hebrew** and **Arabic** — two sister Semitic languages that share roots, structure, and history.

The app is built for two audiences at once:

- 🇮🇱 **Hebrew speakers** learning **Arabic**
- 🇸🇦 **Arabic speakers** learning **Hebrew**

The interface dynamically adapts to the learner's native language: a Hebrew speaker sees the UI in Hebrew while studying Arabic content, and vice versa — making the learning curve gentle and intuitive from the very first screen.

---

## ✨ Key Features

- 🔤 **Letters Module** — Learn the alphabet of the target language with native-script display
- 🔢 **Numbers Module** — Counting, digits, and number-word recognition
- 🔄 **Bidirectional UI** — Full Hebrew↔Arabic interface flip based on learner profile
- 📱 **Mobile-First** — Designed as a native-feeling app experience (single-column, app-shell layout)
- 🎨 **Custom Design System** — Hand-built tokens for color, typography, spacing, and motion
- ♿ **Accessibility-Conscious** — Keyboard focus states, reduced-motion support, and high-contrast text

## 🎨 Design Language

- **Brand Teal** (`#0E7C7B`) — primary action color
- **Accent Terracotta** (`#D97742`) — warmth and Middle-Eastern feel
- **Accent Saffron** (`#F4B72E`) — highlights and progress
- **Cream Surface** (`#FBF6EC`) — soft background for long study sessions
- **Heebo** for Hebrew typography
- **Noto Sans Arabic** for Arabic typography
